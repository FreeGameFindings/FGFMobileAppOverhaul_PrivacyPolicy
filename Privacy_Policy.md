# Content of the Free Game Findings Application
The content of the application is based on data pulled from Reddit using the Reddit API and consists of shared links/text to/about video game promotions. Content will be provided from the publicly available data from the /r/FreeGameFindings subreddit. The application does not require you to have any account to use it for basic viewing, but the links provided from the data may require such. The goal is to allow you to be redirected to the direct link of the promotions shared in the Free Game Findings post(s).

The posts include discussion or links to video games that can vary in age-range and in content. It is possible you may see inappropriate titles, or thumbnails on occasion, but can filter out NSFW content if you wish. It is also possible that content could be deemed inappropriate. Some content accessible by clicking a link from the application may also be inappropriate.

Some settings you configure in the application will be stored on your local device.
